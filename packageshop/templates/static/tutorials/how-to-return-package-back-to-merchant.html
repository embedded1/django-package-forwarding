{% trans "Introduction" %}

{% blocktrans %} Sometime packages have to be returned - we get that. Therefore, we've made it quick and simple to return your packages to the merchant from our warehouse. That's right, We'll do it for you. You do need to contact the merchant first to ensure it will receive your returned merchandise and to get a prepaid return label so you can avoid paying the shipping costs yourself. Once you've done that, leave the rest to us. {% endblocktrans %}

{% trans "Walkthrough" %}
  • {% trans "Log in to your USendHome control panel and click on the Incoming Packages tab" %}
  • {% trans "Click the drop-down menu next to the desired package and select the Return-to-Merchant option" %}
  • {% trans "Allow the wizard to take you through the Return-to-Merchant Checkout process" %}
    • {% trans "Upload the merchant's prepaid return label if you have one and proceed to the next step" %}
    • {% trans "Enter the merchandise's total value if you don't have prepaid return label and proceed to the next step" %}
    • {% trans "Select a shipping address from your address book or enter the merchant's address manually" %}
    • {% trans "Select your preferred shipping method" %}
    • {% trans "Complete payment" %}
  • {% trans "Enable the Package tracking option in your control panel so we can keep you abreast of your return packages progress" %}
