{% trans "How to consolidate multiple parcels into one shipment?" %}

{% trans "Introduction" %}

{% blocktrans %} Merchants have a bad habit of shipping merchandise in multiple packages. This is expensive and inefficient. Let us merge all of your goods into one box to be delivered to you in one shipment. We always use the smallest box that fits all of your goods, and if we do need to use a second box, it'll be the smallest, too! Give us a blanket instruction to consolidate every incoming package of yours and enjoy added discounts. Read this tutorial to find out how to do that; keep reading here to set up a one-time Package Consolidation service. {% endblocktrans %}

{% trans "Walkthrough" %}

There are 2 ways to request Package Consolidation for your parcels:

  • Order the Package Consolidation service in advance (20% off retail service price): This method allows you to request the Package Consolidation service in advance for all incoming packages. When this option is selected, we will immediately consolidate your parcels as soon as we receive them.
    • Login to your Control Panel.
    • Scroll down the right part of the page until you see “Order Extra Service in Advance” Within the option, select “Package Consolidation” and click the Save button.
  • Order the Package Consolidation service after we receive your package"

    • {% trans "Click the Consolidate option next to the package you wish to consolidate." %}
    • {% trans "Repeat Step 2 for every package you wish to consolidate." %}
    • {% trans "Click on the Package Consolidation tab on the main control panel page once all the packages have been selected and click on the Consolidate Packages button." %}
